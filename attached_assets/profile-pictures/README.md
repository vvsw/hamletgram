# Character Profile Pictures

This folder contains the profile pictures for all the Hamlet characters.

## How to Add Your Own Images

Simply replace the placeholder files with your own images:

- **hamlet.png** - Hamlet's profile picture
- **ophelia.png** - Ophelia's profile picture
- **claudius.png** - King Claudius's profile picture
- **gertrude.png** - Queen Gertrude's profile picture
- **horatio.png** - Horatio's profile picture
- **polonius.png** - Polonius's profile picture
- **laertes.png** - Laertes's profile picture
- **ghost.png** - Old Hamlet's Ghost profile picture

## Image Requirements

- **Format**: PNG, JPG, or JPEG
- **Recommended Size**: 400x400 pixels (square images work best)
- **File Names**: Keep the exact file names as listed above (including .png extension)

## Quick Steps

1. Find or create images for your characters
2. Rename them to match the file names above
3. Drop them into this folder (replacing the existing files)
4. Refresh your browser to see the changes!

That's it! The app will automatically use your new images.
